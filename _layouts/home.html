Why this exists

The interesting part is operational texture

This is not a generic “my homelab” blog. The useful bit is the interaction between real household infrastructure and coding agents making decisions inside it.

The redesign leans into that angle: editorial typography, operator-console motifs, animated counters, and staged reveals that make the home page feel like a system summary instead of a markdown dump.
