# Homelab notes that stay close to the machine

This site tracks a small home setup built around NurseDroid, a kitchen display, an Orange Pi monitoring node, a Pixel sensor bridge, and Home Assistant.

The useful distinction here is between history and proof. Old build notes matter, but live claims only stay on the page when they were checked again from the current machine.

The result is part build log, part audit trail, and part setup guide for the pieces that are stable enough to explain clearly.
